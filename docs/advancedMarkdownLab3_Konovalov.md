## Сноска

Markdown полезен в разработке[^1].

[^1]: Markdown поддерживается на GitHub, GitLab, в большинстве редакторов кода и систем документирования.

## Alert-блоки

> [!TIP]
> Для экранирования специальных символов в Markdown используйте обратный слэш `\`. Например, чтобы написать *курсив* без форматирования, напишите `\*курсив\*`.

## Escape-символы

- Заголовок без форматирования: `\# Это не заголовок`
- Звёздочка как текст: `\*просто звёздочка\*`

## HTML в Markdown

```
<p style="background-color: #945a5aff; padding: 15px; border-radius: 8px; border-left: 4px solid #c91616ff;">
Этот параграф оформлен с помощью <u>HTML-тегов</u> внутри Markdown-документа.
</p>
```
